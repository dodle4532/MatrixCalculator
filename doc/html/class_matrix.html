◆ pow()

Matrix Matrix::pow ( int degree)

Method to raise quadratic matrix to any n degree.

Parameters
nCan be natural, or 0, or -1
Returns
Doesn't change matrix, returns powed matrix, if n = 0 then E, if n = -1 then reversed matrix

Example using

int main() {
std::vector<std::vector<float>> vec = {{1, 0},
{0, 8}};
Matrix m(vec);
std::cout << m.pow(2) << std::endl;
// 1 0
// 0 64
std::cout << m.pow(0) << std::endl;
// 1 0
// 0 1
std::cout << m.pow(-1) << std::endl;
// 1 0
// 0 0.1
return 0;
}

Matrix & Matrix::transpose ( )

Method to transpose matrices.

Changes rows and columns with each others(can be done only with quadratic matrix)

It changes matrix itself and returns result

Example using

{4.2, 8}};

//m.transpose();

// 0.5 8
return 0;
}
